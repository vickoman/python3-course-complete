
### A) INT
Para numeros enteros

Para numeros decimales

### C) Complex
Para numeros complejos, osea de la forma, (A+Bj) donde la unidad imaginaria esta representada por la j


```python
c = 1 + 2j
```


```python
c, type(c)
```




    ((1+2j), complex)




```python
parte_real = c.real
parte_imaginaria = c.imag
```


```python
parte_real, parte_imaginaria
```
